# About 
Spring Boot Websocket example using [STOMP](https://en.wikipedia.org/wiki/Streaming_Text_Oriented_Messaging_Protocol)

# Requirements

1. Java 1.8
1. Browser IE8+ 

# To run:

```
mvn spring-boot:run
```

# Open browser:
  
```
http://localhost:8080/
```  
connect, enter name, see greeting.
